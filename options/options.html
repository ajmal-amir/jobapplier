📖 EmailJS Setup Guide (click to expand)
  1. Go to emailjs.com and sign up for a free account
  2. Click Email Services → Add a service (choose Gmail)
  3. Copy your Service ID (looks like: service_abc123)
  4. Click Email Templates → Create a new template
  5. In the template, add these variables: {{subject}}, {{html_body}}
  6. Copy your Template ID (looks like: template_xyz789)
  7. Go to Account → API Keys → copy your Public Key
  8. Paste all three values below and click Save
